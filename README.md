# Mocca demo tests project.

#### Info
This project uses CodeceptJS with Playwright.
Playwright can be easily changed on webDriver (Selenium) or Puppeteer.

#### Installation
The project needs npm to be installed. It was tested on npm 6.14.8
1. `cd` to the project root
2. install all dependencies with you favorite terminal and `npm install -g`

#### Run
run tests with `npm run codeceptjs`
